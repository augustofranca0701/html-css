Tags mais Comuns em CSS

Aqui você encontrará exemplos de algumas das tags mais comuns em CSS.

Tag color


    color: red;

A tag color define a cor do texto.

Tag background-color


    background-color: #F0F8FF;

A tag background-color define a cor do plano de fundo.

Tag font-family


    font-family: Arial, sans-serif;

A tag font-family define a fonte utilizada para o texto.

Tag font-size


    font-size: 16px;

A tag font-size define o tamanho da fonte utilizada para o texto.

Tag text-align


    text-align: center;

A tag text-align define a posição do texto dentro de um elemento.
